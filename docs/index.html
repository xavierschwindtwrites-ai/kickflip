What KickFlip does

Per-backer margin calculator

See exactly what you net after Kickstarter fees, printing, and shipping before you launch.

Three-scenario modeler

Plan for conservative, expected, and breakout funding — and stress test your plan.

Shipping planner

Know your total shipping exposure across every region before you set your goal.

Stretch goal ladder

Build stretch goals that are actually funded by the backers who unlock them.

Launch readiness check

A checklist that pulls from your real data to tell you if you’re ready.

Post-campaign retrospective

Compare actuals to projections and carry lessons into your next campaign.

Getting started

Mac

  1. Download the .dmg file
  2. Open it and drag KickFlip to your Applications folder
  3. Try to open KickFlip — macOS will show a warning
  4. Open Terminal (press Command + Space, type Terminal, hit Enter)
  5. Paste this command and press Enter:
    xattr -cr /Applications/KickFlip.app
  6. Double-click KickFlip in Applications — it will open normally

You only need to do this once. After this KickFlip opens normally every time. This is required because KickFlip is not yet code-signed with an Apple Developer certificate.

Windows

  1. Download the .exe installer
  2. Run it — click More info then Run anyway if prompted
  3. Launch KickFlip from the Start menu
  4. Start planning your campaign

Built for authors, by an author

KickFlip is a free community tool. I built it while planning my own Kickstarter campaign because I couldn’t find anything that answered the questions I actually had. It will always be free.
